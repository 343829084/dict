dict
====

linux命令行下翻译工具

---
#### 安装

```
wget https://raw.githubusercontent.com/wufeifei/dict/master/dict.py
sudo mv ./dict.py /usr/bin/dict
sudo chmod +x /usr/bin/dict
```

#### 使用

```
$ dict test
###################################
#  test 测试 (U: tɛst E: test )
#  n. 试验；检验
#  vt. 试验；测试
#  vi. 试验；测试
#  n. (Test)人名；(英)特斯特
###################################
```
